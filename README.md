# my-project

> 资管 PC交易


## 注意事项
```
    标准版修改问题打包,然后上传ZIP包。防止后面重复问题。
    其他特殊项目(改动项目、新增页面，删除功能等)新建分支，以项目名称命名分支，减少仓库数量。

    如需复制项目，只需将dist.zip下载。
    如修改接口地址，只需修改如下文件的接口地址即可。
    dist
     ├──serverconfig.json

    如特殊项目自行拉取所有代码。新建项目。
    只拉取dist文件的，无需拉取所有代码。

```

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
